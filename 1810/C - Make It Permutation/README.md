This operation costs $$$d$$$.</li></ul><p>You want to make the final array a permutation of <span class="tex-font-style-bf">any</span> positive length. Please output the minimum cost of doing that. For example, $$$[2,3,1,5,4]$$$ is a permutation, but $$$[1,2,2]$$$ is not a permutation ($$$2$$$ appears twice in the array), and $$$[1,3,4]$$$ is also not a permutation ($$$n=3$$$ but there is $$$4$$$ in the array).</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases.
